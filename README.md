# 📋 Arbeitsagentur Unternehmensdarstellung – Dialysezentrum Heidelberg

## Übersicht

Dieses Paket enthält alle vorbereiteten Inhalte und Materialien, um die **Unternehmensdarstellung** auf der Bundesagentur für Arbeit für das **Dialysezentrum Heidelberg** (Zentrum für Nieren- und Hochdruckkrankheiten Bommer - Ehehalt - Ziegler) auszufüllen.

**Ziel:** Pflegekräfte und MFA für die Dialyse gewinnen.

## 🚀 Schritt-für-Schritt Anleitung

### 1. Formular öffnen
Melden Sie sich bei der Arbeitsagentur an und navigieren Sie zu:
**Profilübersicht → Stellenangebote → Unternehmensdarstellung**

### 2. Inhalte-Leitfaden öffnen
Öffnen Sie `formular-inhalte.html` in Ihrem Browser (Doppelklick auf die Datei).

### 3. Titelbild hochladen
1. Öffnen Sie `titelbild.html` im Browser
2. Drücken Sie `Win + Shift + S` (Windows Screenshot-Tool)
3. Wählen Sie den Banner-Bereich aus
4. Speichern Sie den Screenshot
5. Laden Sie das Bild im Formular als **Titelbild** hoch

### 4. Logo hochladen
1. In `formular-inhalte.html` unter "Titelbild und Logo"
2. Rechtsklick auf das dunkle Logo → "Bild speichern unter"
3. Laden Sie das Logo im Formular als **Firmenlogo** hoch

### 5. Eckdaten ausfüllen
Kopieren Sie die Werte aus dem Leitfaden:
- **Hauptsitz:** Heidelberg
- **Gründungsjahr:** 1971
- **Betriebsgröße:** 11–50 Mitarbeitende
- **Tätigkeitsfelder:** Dialyse, Nephrologie, Gesundheitswesen, Pflege, Ambulante Versorgung

### 6. Beschreibung (Über uns)
Klicken Sie auf den Text im Leitfaden → wird automatisch kopiert → im Formular einfügen (Strg+V).
Der Text ist auf **unter 2000 Zeichen** optimiert.

### 7. Vorteile für Mitarbeitende
**Im Formular anklicken:**
- ✅ Flexible Arbeitszeiten
- ✅ Weiterbildungen
- ✅ Betriebliche Altersvorsorge
- ✅ ÖPNV-Ticket / Job-Ticket
- ✅ Parkmöglichkeiten
- ✅ Gute Verkehrsanbindung
- ✅ Mentorenprogramm/Coaching
- ✅ Firmenevents
- ✅ Moderne technische Ausstattung
- ✅ Mitgestaltungsmöglichkeiten

**Eigene Benefits eintippen:**
1. Keine Nachtdienste
2. Keine Sonntagsdienste
3. Keine Bereitschaftsdienste
4. Tiefgarage mitten in der Stadt
5. Strukturierte Dialyse-Einarbeitung

### 8. Kontaktdaten
Klicken Sie auf den Kontakttext im Leitfaden → kopieren → einfügen.
Der Text ist auf **unter 500 Zeichen** optimiert.

### 9. QR-Codes (Bonus)
- Öffnen Sie `qr-bewerbung.html` für eine druckbare A4-Seite
- Die QR-Codes können auch als Firmenfotos hochgeladen werden
- Bewerber scannen den Code → vorausgefüllte Nachricht öffnet sich → nur Name & Telefonnummer einsetzen → absenden
